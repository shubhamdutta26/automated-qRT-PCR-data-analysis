Guidelines for using this application

  • This application uses the Delta-Delta-Cq method for calculating Gene expression.
  • There needs to atleast three Biological replicates and two Technical replicates.
  • "Endogenous Control Gene" & "Reference Condition" names should exactly match the input file.
  • Rename columns as follows: Target gene as "Target" & Biological set as "Condition".
  • Each Sample should have a different name even if they are the same Biological Set.
